\textbf{1. Теорема о произведении абсолютно сходящихся рядов:} \smallskip

Если ряды $\sum a_n$ и $\sum b_n$ абсолютно сходятся, то при любой
нумерации элементов матрицы $C$ ряд $\sum c_n$ абсолютно сходится и

\[\sum^{\infty}_{n = 1} c_n = \sum^{\infty}_{n = 1} a_n *
\sum^{\infty}_{n = 1} b_n.\]

Доказательство: \smallskip

Рассмотрим нумерацию: \smallskip

\[c_1 = a_1 b_1, c_2 = a_1 b_2, c_3 = a_2 b_2, c_4 = a_2 b_1,\]

\[c_5 = a_1 b_3, c_6 = a_2 b_3, c_7 = a_3 b_3, c_8 = a_3 b_2,
c_9 = a_3 b_1, \dots\]

Тогда видно, что 

\[\sum^n_{k = 1} |c_k| \leq \sum^{n^2}_{k = 1} |c_k| = 
\sum^n_{k = 1} |a_k| * \sum^n_{k = 1} |b_k|.\]

Переходя к пределу при $n \to \infty$, получим

\[\sum^{\infty}_{k = 1} |c_k| \leq \sum^{\infty}_{k = 1} |a_k| *
\sum^{\infty}_{k = 1} |b_k| < +\infty,\]

т.е. ряд $\sum c_n$ абсолютно сходится. Тогда любая его перестановка
$\sum c_{\phi(n)}$ абсолютно сходится и

\[\sum^{\infty}_{n = 1} c_{\phi(n)} = \sum^{\infty}_{n = 1} c_n.\]

Осталось в ряду $\sum c_n$ произвести группировку

\[c_1 + (c_2 + c_3 + c_4) + (c_5 + \dots + c_9) + \dots,\]

согласованную с перемножением частных сумм и воспользоваться ранее
доказанными фактами. Таким образом, при любой нумерации элементов
матрицы $C$ получаем ряд, сумма которого

\[\sum^{\infty}_{n = 1} c_{\phi(n)} = \sum^{\infty}_{n = 1} c_n =
\sum^{\infty}_{n = 1} a_n * \sum^{\infty}_{n = 1} b_n.\]
