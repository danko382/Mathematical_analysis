\textbf{1. Признак Абеля:}

1) последовательность $(a_n)$ монотонна и ограничена; \smallskip

2) ряд $\sum b_n$ сходится; \smallskip

Тогда ряд $\sum a_n b_n$ сходится. \smallskip

Доказательство: \smallskip

Пусть $|a_k| \leq M, k \geq 1.$ Запишем условие Коши для сходящегося
ряда $\sum b_n:$

\[\forall \epsilon > 0 \quad \exists n_0 \in N
\quad \forall n \geq n_0 \quad \forall p \in N \quad
\left| \sum^{n + p}_{k = n + 1} b_k \right| < \epsilon.\]

Обозначим 

\[B^n_p = \sum^{n + p}_{k = n + 1} b_k.\]

В этих обозначениях имеем

\[\left| B^n_p \right| < \epsilon, \quad n \geq n_0, \quad p \in N.\]

Применив к сумме

\[\sum^{n + p}_{k = n + 1} a_k b_k\]

преобразование Абеля, получим

\[\sum^{n + p}_{k = n + 1} a_k b_k = a_{n + p} B^n_p - 
\sum^{n + p - 1}_{k = n + 1} (a_{k + 1} - a_k) B^n_k.\]

При $n \geq n_0$ имеем

\[\left| \sum^{n + p}_{k = n + 1} a_k b_k \right| \leq
|a_{n + p}||B^n_p| + \sum^{n + p - 1}_{k = n + 1} |a_{k + 1} - a_k|
|B^n_k| \leq \] 

\[\leq M \epsilon + \epsilon \sum^{n + p - 1}_{k = n + 1}
|a_{k + 1} - a_k|.\]

В силу монотонности и ограниченности последовательности $(a_n$)

\[\sum^{n + p - 1}_{k = n + 1} |(a_{k + 1} - a_k)| = |a_{n + p} - a_{n + 1}|
\leq 2M\]

и

\[\left| \sum^{n + p}_{k = n + 1} a_k b_k \right| < M \epsilon
+ \epsilon 2M = 3M \epsilon.\]

По критерию Коши ряд $\sum a_n b_n$ сходится. \bigskip

\textbf{2. Признак Дирихле:} \smallskip

1) последовательность $(a_n)$ монотонная и бесконечно малая;

2) последовательность $B_n = \sum^n_{k = 1} b_k$ ограничена. \smallskip

Тогда ряд $\sum a_n b_n$ сходится.\smallskip

Доказательство: \smallskip

Согласно условиям теоремы

\[a_n B_n = o(1)O(1) = o(1).\]

В силу теоремы о равносходимости рядов, связанных преобразованием Абеля,
ряды $\sum a_n b_n$ и $\sum B_n(a_{n + 1} - a_n)$ ведут себя одинаково.
К исследованию сходимости вроторого из этих рядов применим критерий
Коши. \smallskip

Пусть $|b_k| \leq M, k \geq 1.$ Возьмём произвольное $\epsilon > 0$
$\exists n_0 \quad \forall n \geq n_0 \quad |a_n| < \epsilon$ (это
возможно поскольку $a_n = o(1)$). \smallskip

При $n \geq n_0$ будем иметь оценку

\[\left| \sum^{n + p}_{k = n + 1} (a_{k + 1} - a_k)B_k \right|
\leq \sum^{n + p}_{k = n + 1} |(a_{k + 1} - a_k)||B_k| \leq M
\sum ^{n + p}_{k = n + 1}|(a_{k + 1} - a_k)|.\]

Так как последовательность $(a_n)$ монотонна, то разности $a_{k + 1} -
a_k$ одного знака и поэтому

\[\sum^{n + p}_{k = n + 1}|(a_{k + 1} - a_k)| = \left| \sum^{n + p}_{k = n + 1}
(a_{k + 1} - a_k) \right| = |a_{n + p + 1} - a_{n + 1}| \leq\]


\[|a_{n + p + 1}| + |a_{n + 1}| < 2 \epsilon.\]

Соединяя все оценки, получим

\[\left| \sum^{n + p}_{k = n + 1} (a_{k + 1} - a_k)B_k \right| <
2 M \epsilon\]

$\forall n \geq n_0$ и $\forall p \geq 1$, что по критерию Коши 
эквивалентно сходимости ряда $\sum(a_{n + 1} - a_n)B_n.$ Теорема
доказана. \bigskip

\textbf{3. Признак Лейбница:} \smallskip

Пусть последовательность $(a_n)$ монотонная и бесконечно малая. Тогда 
ряд $\sum (-1)^{n - 1} a_n$ сходится. \smallskip

Доказательство: \smallskip

Нужно положить $b_n = (-1)^{n - 1}$ и воспользоваться признаком Дирихле.
