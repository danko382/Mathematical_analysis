\textbf{1. Дифференциал функции первого порядка:} \smallskip

Пусть функция $f$ дифференцируема в точке $\overline{x}_0$. определим
новую функцию $df(\overline{x}_0) : R^m \to R,$ полагая для 
лубого $\overline{h} \in R^m$

\[df(\overline{x}_0)(\overline{h}) = \sum^m_{k = 1} \frac{df}{dx_k}
(\overline{x}_0)h_k.\]

Указанная функция называется дифференциалом функции $f$ в точке
$\overline{x}_0$. \bigskip

\textbf{2. Дифференциал функции второго порядка:} \smallskip

Пусть функция $f$ 2"=дифференцируема в точке $\overline{x}_0$.
Для каждого $\overline{h} \in R^m$ определим в окрестности
$O(\overline{x}_0)$ точки $\overline{x}_0$ функцию 
$d_{\overline{h}}:R^m \to R,$ полагая для каждого $\overline{x} \in 
O(\overline{x}_0)$

\[d_{\overline{h}}f(\overline{x}) = df(\overline{x})(\overline{h}).\]

Дифференциалом 2"=го порядка функции $f$ в точке $\overline{x}_0$
называется функция $d^2 f(\overline{x}_0):R^m \to R$ такая, что
$\forall \overline{h} \in R^m$

\[d^2f(\overline{x}_0)(\overline{h}) = d(d_{\overline{h}}f)(\overline{x}_0)
(\overline{h})\]

Дифференциал произвольного порядка определяется по индукции.
