\textbf{1. Преобразование Абеля:} \smallskip

Пусть $B_n = \sum^n_{k = 1} b_k, n \geq 1.$ Тогда справедлива формула

\[\sum^n_{k = 1} a_k b_k = a_n B_n - \sum^{n - 1}_{k = 1}
(a_{k + 1} - a_k)B_k \quad \quad (*)\] \smallskip

Доказательство: \smallskip

Принимая во внимание очевидные равенства 

\[b_1 = B_1, b_2 = B_2 - B_1, \dots , b_n = B_n - B_{n - 1},\]

получим

\[\sum^n_{k = 1} a_k b_k = a_1 B_1 + a_2(B_2 - B_1) + \dots +
a_n (B_n - B_{n - 1}) = \] 

\[= B_1(a_1 - a_2) + B_2(a_2 - a_3) + \dots 
+B_{n - 1}(a_{n - 1} - a_n) + B_n a_n =\] 

\[= \sum^{n - 1}_{k = 1}
B_k(a_k - a_{k + 1}) + B_n a_n,\]

что эквивалентно равенству (*). \bigskip

\textbf{2. Теорема о равносходимости рядов, связанных преобразованием
Абеля:} \smallskip

Пусть $B_n = \sum^n_{k = 1} b_k,$ и последовательность $(a_n B_n)$
сходится. Тогда ряды $\sum a_n b_ n$ и $\sum B_n (a_{n + 1} - a_n)$
ведут себя одинаково.
